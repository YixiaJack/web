Reuniverse is a professionally oriented concept website for an open-source platform where public ideas become planets in a shared universe. The site presents the project as both a speculative interface and a practical collaboration tool: each idea is assigned a permanent real celestial body, receives public Stars, advances through civilization stages, and opens into a GitHub-connected workspace. The website is organized across linked pages. The home page introduces the slogan, core mechanisms, audio atmosphere, and visual identity. The concept page explains body assignment, civilization stages, example planets, and the philosophy behind attention becoming civilization. The demo page lets visitors publish an idea draft, assign a celestial body through JavaScript, switch between Surface and Workspace summary views, star once as a visitor, and simulate public attention growth. The separate workspace page contains the full Work Tracker with lightweight cards, drag-and-drop pipelines, issue creation, editing, story points, and deletion. The design uses NASA-inspired archive imagery, restrained product-interface colors, and clear semantic sections so the idea feels credible after the class rather than like a temporary school exercise.
